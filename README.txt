
Experimental appleseed maya integration plugin.
-----------------------------------------------


Setup:
------

    1. set env vars:
        export MAYA_PLUG_IN_PATH=${BUILD_DIR}/src
        export MAYA_SCRIPT_PATH=${THIS_DIR}/appleseed-maya/scripts
        export PYTHONPATH=$PYTHONPATH:${APPLESEED_DIR}/lib/python2.7:${THIS_DIR}/scripts
        export XBMLANGPATH==${THIS_DIR}/icons

    2. launch Maya & load plugin.
